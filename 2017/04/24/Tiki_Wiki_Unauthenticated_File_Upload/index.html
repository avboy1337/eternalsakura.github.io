Tiki Wiki <=15.1 - Unauthenticated File Upload

漏洞原理

利用Tiki利用Tiki Wiki<=15.1中的文件上传漏洞，
漏洞来自tiki的第三方组件ELFinder -version 2.0
该组件带有一个默认的示例页面：包括上传、删除、重命名、创建目录等
并且该页面不强制进行文件扩展名、内容类型等验证。
因此，未经身份验证的用户也可以上传PHP文件。

3.检测是否上传成功
